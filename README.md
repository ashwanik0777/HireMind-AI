# HireMind AI

HireMind AI is an AI-powered resume and portfolio analysis platform designed to help students and freshers improve their ATS score, optimize resumes, and enhance their chances of getting shortlisted for tech roles.

## Tech Stack

* Frontend: Next.js
* Styling: Tailwind CSS
* Backend: Node.js + Express.js
* Database: PostgreSQL
* AI API: Gemini API
* Authentication: JWT
* Deployment: Vercel + Render
